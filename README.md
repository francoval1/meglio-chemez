## Meglio Chemez Legal Hub

Aplicación web creada con **Vite + React + TypeScript + Tailwind + shadcn-ui** para el estudio jurídico Meglio Chemez.

### Requisitos previos

- **Node.js** (versión 18 o superior recomendada)
- **npm** (incluido con Node)

### Instalación y ejecución en localhost

1. Abrir una terminal y ubicarse en la carpeta del proyecto:

   ```sh
   cd megliochemez-legal-hub-main
   ```

2. Instalar dependencias:

   ```sh
   npm install
   ```

3. Levantar el servidor de desarrollo:

   ```sh
   npm run dev
   ```

4. Abrir el navegador en:

   ```text
   http://localhost:8080
   ```

### Scripts disponibles

- **`npm run dev`**: inicia el servidor de desarrollo.
- **`npm run build`**: genera la build de producción.
- **`npm run preview`**: sirve la build de producción localmente.
- **`npm run lint`**: ejecuta ESLint.
- **`npm test`**: ejecuta los tests con Vitest.

### Tecnologías

- **Vite**
- **React 18**
- **TypeScript**
- **Tailwind CSS**
- **shadcn-ui / Radix UI**
